## 🆘 Ajuda Rápida

### Vejo erro no site?

1. Abra o Console (F12)
2. Verifique se há erros vermelhos
3. Se sim, veja os logs nos terminais

### API não responde?

```powershell

```

### Preciso reiniciar tudo?

Feche os terminais e execute:

```powershell
.\start-tudo.bat
```
